Introduction¶

Thank you for deciding to check out this library!

This package is designed for effective and speedy materials analysis. It was developed at the National Institute of Standards and Technology in Gaithersburg, MD.

To get started, import the intelligent_materials package and ensure that you have all of the necessary dependencies. This project is also available on PyPI as Intelligent Material Design (https://pypi.python.org/pypi/Intelligent_Material_Design).

Included are: Graph Suite (graph_suite.py), the Data Extractor (parser.py), Material Analytics (material_analytics.py), and Optimization Suite (optimization_suite.py).
